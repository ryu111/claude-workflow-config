## Quick Reference

### 風控選擇指南

| 情境 | 推薦風控類型 | 原因 |
|------|--------------|------|
| 新手/簡單策略 | 靜態 | 容易理解和執行 |
| 波動市場 | 動態 | 自適應市場變化 |
| 專業交易 | 混合 | 基礎穩定 + 動態調整 |
| 高槓桿交易 | 嚴格動態 | 避免爆倉 |

### 核心公式

```python
# 部位大小 = 風險金額 / 單位風險
position_size = (account * risk_pct) / abs(entry - stop_loss)

# 槓桿 = 部位價值 / 帳戶淨值
leverage = (position_size * entry_price) / account

# 強平價格（多單）
liquidation_price = entry * (1 - 1/leverage + maintenance_margin)
```

### 建議參數

| 參數 | 保守 | 適中 | 積極 |
|------|------|------|------|
| 單筆風險 | 0.5% | 1-2% | 2-3% |
| 最大槓桿 | 3x | 5x | 10x |
| 最大持倉 | 3 筆 | 5 筆 | 7 筆 |
| 日損失上限 | 2% | 5% | 10% |
| 最大回撤 | 10% | 20% | 30% |
