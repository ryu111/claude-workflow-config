# Emotional Design 情感設計

基於 Don Norman 的情感設計理論，創造與使用者建立情感連結的產品體驗。

---

## Don Norman 三層次理論

### 概述

```
┌─────────────────────────────────┐
│     Reflective 反思層            │  ← 意義、記憶、自我形象
│  "這代表我是什麼樣的人"           │
├─────────────────────────────────┤
│     Behavioral 行為層            │  ← 功能、可用性、效率
│  "用起來感覺如何"                 │
├─────────────────────────────────┤
│     Visceral 本能層              │  ← 第一印象、感官反應
│  "看起來/感覺起來如何"            │
└─────────────────────────────────┘
```

---

## 1. Visceral Design 本能層設計

> **第一印象，在理性思考之前的直覺反應。**

### 觸發因素

| 感官 | 設計元素 |
|------|----------|
| 視覺 | 色彩、形狀、對比、動畫 |
| 聽覺 | 音效、音樂、通知聲 |
| 觸覺 | 觸感回饋、震動、手勢 |

### 設計原則

```
✓ 視覺吸引力
  - 和諧的色彩搭配
  - 精緻的圖示和插圖
  - 流暢的動畫效果

✓ 感官愉悅
  - 令人滿意的互動回饋
  - 柔和的過渡效果
  - 適當的聲音設計

✓ 品牌印象
  - 獨特的視覺識別
  - 一致的設計語言
  - 專業的呈現品質
```

### 範例

```
✅ 良好的本能設計

載入動畫：
○ → ◐ → ◑ → ● → ✓
流暢、有節奏感、完成時有滿足感

按鈕回饋：
[送出] → [送出 ↻] → [✓ 已送出]
狀態變化清晰，完成有正面感受

❌ 缺乏本能設計

[送出] → ... → [完成]
無過渡、無動畫、無愉悅感
```

### 色彩心理學

| 色彩 | 情感聯想 | 適用場景 |
|------|----------|----------|
| 藍色 | 信任、專業、冷靜 | 企業、金融、科技 |
| 綠色 | 自然、成長、成功 | 環保、健康、財務正向 |
| 紅色 | 緊急、熱情、警示 | CTA、錯誤、促銷 |
| 橘色 | 活力、友善、溫暖 | 社交、食品、娛樂 |
| 紫色 | 創意、奢華、神秘 | 美妝、高端品牌 |
| 黃色 | 樂觀、注意、警告 | 提醒、促銷、兒童 |

## 2. Behavioral Design 行為層設計

> **使用過程中的體驗，專注於功能性和可用性。**

### 核心要素

```
功能性：是否能完成任務？
可用性：是否容易使用？
效率性：是否快速有效？
可靠性：是否穩定無錯？
```

### 設計原則

```
✓ 流暢的互動
  - 回應迅速 (<100ms 感覺即時)
  - 操作直覺
  - 回饋清楚

✓ 掌控感
  - 可預測的結果
  - 可逆的操作
  - 清楚的狀態

✓ 效率
  - 最少步驟完成任務
  - 快捷鍵和手勢
  - 智慧預設
```

### 微互動設計

```
觸發器 (Trigger)
    ↓
規則 (Rules)
    ↓
回饋 (Feedback)
    ↓
循環 (Loops & Modes)
```

#### 範例：按讚互動

```
觸發：點擊愛心圖示
規則：切換按讚狀態、更新計數
回饋：
  ♡ → ❤️ (顏色變化)
  + 縮放動畫
  + 數字 +1 動畫
  + (可選) 震動回饋
循環：持續顯示狀態、可再次點擊取消
```

### 回饋類型

| 類型 | 用途 | 範例 |
|------|------|------|
| 視覺 | 狀態變化 | 顏色、大小、圖示變化 |
| 動態 | 過程指示 | Loading、進度條 |
| 聲音 | 引起注意 | 通知音、完成音 |
| 觸覺 | 確認動作 | 震動回饋 |

## 3. Reflective Design 反思層設計

> **使用者如何看待產品對自己的意義，關乎自我形象和記憶。**

### 核心要素

```
自我形象：使用這個產品代表我是什麼樣的人？
社會認同：這符合我想要的形象嗎？
個人意義：這對我有什麼特別意義？
記憶：我會記得這個體驗嗎？
```

### 設計原則

```
✓ 品牌價值觀
  - 明確的品牌定位
  - 一致的品牌體驗
  - 符合目標使用者價值觀

✓ 個人化
  - 允許自訂外觀
  - 記住偏好
  - 個人化內容

✓ 社群認同
  - 使用者社群
  - 分享功能
  - 身份標籤/徽章
```

### 範例

```
Apple 產品：
- 本能層：簡潔美觀的設計
- 行為層：直覺的操作體驗
- 反思層：「我是注重設計和品質的人」

MUJI 產品：
- 本能層：素雅簡約的外觀
- 行為層：實用功能
- 反思層：「我崇尚簡單生活」

遊戲成就系統：
┌─────────────────────────────────┐
│ 🏆 恭喜！你獲得了「早起鳥」徽章   │
│                                 │
│ 連續 30 天在早上 7 點前打開 App  │
│                                 │
│ [分享到社群]                     │
└─────────────────────────────────┘
↑ 建立自我認同和成就感
```

## 微文案 (Microcopy) 設計

> **介面中的小段文字，對情感體驗有巨大影響。**

### 原則

```
✓ 人性化語調
✓ 清晰具體
✓ 正面積極
✓ 符合品牌個性
```

### 範例對比

| 情境 | 冷淡版 | 人性化版 |
|------|--------|----------|
| 空狀態 | "無資料" | "還沒有任何項目，來建立第一個吧！" |
| 載入中 | "載入中..." | "正在為你準備內容..." |
| 錯誤 | "錯誤 404" | "哎呀，這個頁面去度假了" |
| 成功 | "完成" | "太棒了！你做到了！" |
| 表單提示 | "輸入 Email" | "你的 Email（我們會保密）" |

### 按鈕文案

```
❌ 模糊
[確定] [取消] [是] [否]

✅ 明確
[刪除專案] [保留專案]
[儲存變更] [放棄變更]
[訂閱電子報] [不，謝謝]
```

### 錯誤訊息

```
❌ 令人沮喪
"輸入無效"
"操作失敗"

✅ 有幫助且友善
┌─────────────────────────────────┐
│ 😅 密碼需要再強一點              │
│                                 │
│ 請加入：                        │
│ ✓ 至少 8 個字元                 │
│ ✗ 一個大寫字母                  │
│ ✗ 一個數字                      │
└─────────────────────────────────┘
```

## 品牌個性塑造

### 定義品牌個性

```
維度：
1. 正式 ←───────→ 休閒
2. 嚴肅 ←───────→ 幽默
3. 權威 ←───────→ 親切
4. 傳統 ←───────→ 創新
```

### 品牌個性範例

| 品牌類型 | 語調 | 範例文案 |
|----------|------|----------|
| 專業金融 | 正式、信任 | "您的交易已安全處理" |
| 社交媒體 | 輕鬆、活潑 | "分享你今天的心情吧！" |
| 遊戲 | 幽默、激勵 | "準備好大顯身手了嗎？" |
| 健康 | 溫暖、支持 | "每一小步都是進步 💪" |

### 一致性檢查

```
✓ 所有文案符合品牌語調
✓ 視覺風格一致
✓ 互動方式一致
✓ 跨平台體驗一致
```

## Dark Patterns 警示

> **避免利用心理學原理操縱使用者的不道德設計。**

### 常見 Dark Patterns

| 名稱 | 說明 | 範例 |
|------|------|------|
| Bait and Switch | 誘導後轉換 | 免費試用 → 強制訂閱 |
| Confirmshaming | 羞恥確認 | "不，我不想省錢" |
| Hidden Costs | 隱藏費用 | 結帳時突然出現運費 |
| Misdirection | 誤導 | 突出錯誤選項 |
| Privacy Zuckering | 隱私陷阱 | 預設公開所有資訊 |
| Roach Motel | 進易出難 | 一鍵訂閱、難以取消 |
| Trick Questions | 誘導問題 | 雙重否定句 |

### 道德設計原則

```
✓ 透明：清楚告知使用者會發生什麼
✓ 尊重：尊重使用者的時間和注意力
✓ 誠實：不使用誤導性語言或設計
✓ 控制：讓使用者輕鬆取消或離開
✓ 公平：不利用認知偏見操縱決策
```

### 正面範例

```
✅ 道德的取消訂閱

┌─────────────────────────────────┐
│ 要取消訂閱嗎？                   │
│                                 │
│ 我們很遺憾看到你離開。           │
│                                 │
│ [暫停訂閱]  [確定取消]           │
│                                 │
│ 你可以隨時重新訂閱               │
└─────────────────────────────────┘

↑ 簡單、明確、無壓力
```

## 情感設計檢查清單

### 本能層 (Visceral)

- [ ] 視覺設計專業美觀
- [ ] 色彩運用符合情感目標
- [ ] 動畫流暢且有目的
- [ ] 整體品質感良好

### 行為層 (Behavioral)

- [ ] 操作直覺易學
- [ ] 回饋即時清楚
- [ ] 流程高效無阻
- [ ] 錯誤容易修復

### 反思層 (Reflective)

- [ ] 品牌個性清晰
- [ ] 使用者有掌控感
- [ ] 創造正面記憶
- [ ] 符合目標使用者價值觀

### 微文案

- [ ] 語調符合品牌個性
- [ ] 文案清晰具體
- [ ] 錯誤訊息友善有幫助
- [ ] 空狀態有引導

### 道德

- [ ] 無 Dark Patterns
- [ ] 尊重使用者選擇
- [ ] 操作透明可預測
- [ ] 容易取消/退出

```
└── 更大的錯誤容忍度

```
