## 詳細產業指南

### 金融科技 (FinTech)

**核心需求**：信任、安全、專業、易用

**推薦組合**：

| 定位 | 風格組合 | 配色 | 字體 |
|------|----------|------|------|
| 保守型 | Corporate Clean | #1e3a5f + #ffffff | Inter + System |
| 現代型 | Clean Minimal + Glass | #2563eb + 漸層 | SF Pro + Inter |
| 年輕型 | Soft Minimal | 紫藍漸層 | Outfit + DM Sans |

**避免**：
- Neubrutalism（太激進）
- Maximalist（太混亂）
- 過多動效（分散注意力）

---

### SaaS 產品

**核心需求**：效率、清晰、專業、可擴展

**推薦組合**：

| 階段 | 風格組合 | 配色 | 理由 |
|------|----------|------|------|
| MVP | Clean Minimal | 單色 + 強調 | 快速開發 |
| 成長期 | Minimal + Cards | 品牌色系統 | 可擴展 |
| 成熟期 | Design System | 完整 Token | 一致性 |

**最佳實踐**：
- 儀表板：數據密度適中
- 表單：清晰分組
- 導航：固定側邊欄

---

### 電商平台

**核心需求**：轉換、信任、產品展示、快速結帳

**推薦組合**：

| 品類 | 風格組合 | 配色 | 重點 |
|------|----------|------|------|
| 時尚 | Editorial + Bold | 黑白 + 強調 | 產品為主 |
| 3C | Tech Minimal | 深色 + 亮點 | 規格清晰 |
| 生鮮 | Organic + Nature | 綠色系 | 新鮮感 |
| 奢侈 | Japanese + Art Deco | 黑金白 | 質感 |

**CTA 優化**：
- 購買按鈕：高對比、固定位置
- 價格：清晰、無隱藏費用
- 庫存：緊迫感但不壓迫

---

### 醫療健康

**核心需求**：信任、安全、無障礙、專業

**法規考量**：
- WCAG AAA 強制（政府醫療）
- 字體 ≥ 16px（建議 18px）
- 對比度 ≥ 7:1（高齡用戶）

**推薦組合**：

| 類型 | 風格組合 | 配色 | 特殊需求 |
|------|----------|------|----------|
| 診所 | Soft Minimal | 藍綠白 | 高無障礙 |
| 心理 | Organic Soft | 紫藍柔和 | 舒緩感 |
| 健身 | Bold Dark | 橙黑 | 動感 |
| 長照 | Accessible | 高對比 | 大字體 |

---

## 使用方式

```
1. 查找產品類型所在表格
2. 記錄主推風格和備選風格
3. 前往對應風格群組檔案查看細節
4. 選擇配色和字體
5. 使用 decision-rules.md 驗證選擇
```
