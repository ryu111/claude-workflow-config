# 進階色彩理論

專業的色彩運用原則，創造視覺和諧且符合品牌調性的介面。

---

## 色彩基礎

### 色彩屬性

```
色相 (Hue)：色彩的種類（紅、藍、綠...）
飽和度 (Saturation)：色彩的純度（鮮豔 ↔ 灰暗）
明度 (Lightness/Value)：色彩的亮度（亮 ↔ 暗）
```

### 色彩模型

| 模型 | 用途 | 說明 |
|------|------|------|
| RGB | 螢幕顯示 | 加色混合 |
| HSL/HSB | 設計工具 | 更直覺的調整 |
| HEX | 網頁開發 | RGB 的十六進位表示 |
| OKLCH | 現代設計 | 感知均勻的色彩空間 |

## 60-30-10 法則

### 核心原則

```
┌─────────────────────────────────────┐
│                                     │
│     60% 主色（Dominant）             │
│     背景、大面積                     │
│                                     │
├─────────────────────────────────────┤
│  30% 次色（Secondary）               │
│  卡片、區塊、導航                    │
├───────────────────┬─────────────────┤
│ 10% 強調色        │  CTA、連結       │
│ (Accent)          │  重點元素        │
└───────────────────┴─────────────────┘
```

### 應用範例

```css
:root {
  /* 60% - 主色（背景） */
  --color-dominant: #ffffff;      /* 淺色主題 */

  /* 30% - 次色（結構） */
  --color-secondary: #f8f9fa;     /* 區塊背景 */
  --color-secondary-dark: #e9ecef; /* 分隔線 */

  /* 10% - 強調色（行動） */
  --color-accent: #2563eb;        /* CTA、連結 */
  --color-accent-hover: #1d4ed8;
}
```

### 實際佈局

```
淺色主題範例：

┌─────────────────────────────────┐
│ Header (#ffffff)                │ ← 60%
├───────────┬─────────────────────┤
│  Sidebar  │  Main Content       │
│ (#f8f9fa) │  (#ffffff)          │ ← 60%
│           │                     │
│  30% ↓    │  [主要按鈕]         │ ← 10%
│           │  (#2563eb)          │
├───────────┴─────────────────────┤
│ Footer (#f8f9fa)               │ ← 30%
└─────────────────────────────────┘
```

## 色彩和諧

### 互補色 Complementary

```
色輪上 180° 對立的顏色
高對比、高衝擊力

     🔴 紅
      ↕
     🟢 青綠

用途：需要強烈對比的場景
注意：避免大面積直接相鄰
```

### 類似色 Analogous

```
色輪上相鄰的 2-3 個顏色
和諧、統一

  🟡 黃 — 🟠 橙 — 🔴 紅

用途：創造統一感
優點：視覺舒適
```

### 三角配色 Triadic

```
色輪上等距 120° 的三個顏色
平衡、活潑

     🔴 紅
    /    \
   🔵    🟡
  藍      黃

用途：需要多樣性但保持平衡
注意：一個主色，其他兩個輔助
```

### 分裂互補 Split-Complementary

```
一個主色 + 其互補色兩側的顏色
比純互補更柔和

     🔴 紅
    /    \
   🟢    🔵
 青綠    青藍

用途：需要對比但不過於強烈
```

### 方形配色 Square/Tetradic

```
色輪上等距 90° 的四個顏色
豐富但需要平衡

🔴 紅 ─── 🟡 黃
    ╲   ╱
     ╲ ╱
    🟣 ─── 🟢
   紫      綠

用途：複雜的設計系統
注意：需要明確的主次關係
```

## 色彩心理學

### 色彩情感聯想

| 色彩 | 正面聯想 | 負面聯想 | 適用場景 |
|------|----------|----------|----------|
| 🔴 紅 | 熱情、活力、緊急 | 危險、錯誤 | CTA、警告、促銷 |
| 🟠 橙 | 友善、溫暖、活力 | 廉價感 | 社交、娛樂、食品 |
| 🟡 黃 | 樂觀、注意、創意 | 警告 | 提醒、兒童、創意 |
| 🟢 綠 | 自然、成功、安全 | 嫉妒 | 環保、健康、財務 |
| 🔵 藍 | 信任、專業、冷靜 | 冷漠 | 企業、科技、金融 |
| 🟣 紫 | 創意、奢華、神秘 | 不成熟 | 美妝、高端、創意 |
| ⚫ 黑 | 優雅、力量、高端 | 死亡、沉重 | 奢侈品、時尚 |
| ⚪ 白 | 純淨、簡潔、現代 | 空洞、冷 | 極簡、醫療 |

### 產業色彩慣例

```
科技：藍色（信任、專業）
金融：藍、綠（信任、財富）
醫療：藍、綠、白（清潔、專業、自然）
食品：紅、橙、黃（食慾、溫暖）
環保：綠、棕（自然、大地）
奢侈品：黑、金（高端、尊貴）
兒童：多彩、高飽和度（活力、歡樂）
```

## 語意化色彩

### 功能色彩

```css
:root {
  /* 成功 - 綠色 */
  --color-success: #10b981;
  --color-success-light: #d1fae5;
  --color-success-dark: #059669;

  /* 警告 - 黃/橙色 */
  --color-warning: #f59e0b;
  --color-warning-light: #fef3c7;
  --color-warning-dark: #d97706;

  /* 錯誤 - 紅色 */
  --color-error: #ef4444;
  --color-error-light: #fee2e2;
  --color-error-dark: #dc2626;

  /* 資訊 - 藍色 */
  --color-info: #3b82f6;
  --color-info-light: #dbeafe;
  --color-info-dark: #2563eb;
}
```

### 使用原則

```
✓ 語意一致
  成功永遠是綠色，錯誤永遠是紅色

✓ 不只依賴顏色
  配合圖示、文字傳達資訊

✓ 提供足夠對比
  文字和背景至少 4.5:1
```

## 對比度計算

### WCAG 標準

| 類型 | AA 標準 | AAA 標準 |
|------|---------|----------|
| 一般文字 | 4.5:1 | 7:1 |
| 大文字 (18px+) | 3:1 | 4.5:1 |
| UI 元件 | 3:1 | - |

### 計算方式

```
對比度 = (L1 + 0.05) / (L2 + 0.05)

L = 相對亮度
L1 = 較亮顏色
L2 = 較暗顏色

相對亮度公式：
L = 0.2126 × R + 0.7152 × G + 0.0722 × B
（其中 R, G, B 經過 gamma 校正）
```

### 常見配色對比度

```
良好組合：
✓ #000000 on #ffffff → 21:1 (完美)
✓ #1a1a1a on #ffffff → 16.1:1 (優秀)
✓ #374151 on #ffffff → 9.6:1 (優秀)
✓ #2563eb on #ffffff → 5.3:1 (AA 通過)

問題組合：
✗ #9ca3af on #ffffff → 2.8:1 (不足)
✗ #d1d5db on #ffffff → 1.5:1 (嚴重不足)
```

## 漸層設計

### 漸層類型

```css
/* 線性漸層 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 放射漸層 */
background: radial-gradient(circle at center, #667eea 0%, #764ba2 100%);

/* 錐形漸層 */
background: conic-gradient(from 45deg, #667eea, #764ba2, #667eea);
```

### 漸層原則

```
✓ 使用相鄰色相
  避免跨越色輪對面（會產生灰濁過渡）

✓ 控制飽和度變化
  保持飽和度相近，避免突兀

✓ 角度選擇
  135deg / 45deg 常見且美觀
  0deg / 90deg 較為傳統

✓ 考慮文字可讀性
  漸層背景上的文字需要足夠對比
```

### 漸層範例

```css
/* 品牌漸層 */
.gradient-brand {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

/* 柔和漸層 */
.gradient-soft {
  background: linear-gradient(135deg, #ffecd2 0%, #fcb69f 100%);
}

/* 深色漸層 */
.gradient-dark {
  background: linear-gradient(135deg, #1a1a2e 0%, #16213e 100%);
}

/* 多色漸層 */
.gradient-multi {
  background: linear-gradient(
    90deg,
    #ff6b6b 0%,
    #feca57 25%,
    #48dbfb 50%,
    #ff9ff3 75%,
    #ff6b6b 100%
  );
}
```

## 數據視覺化色彩

### 分類色彩

```
用於區分不同類別的數據

原則：
✓ 視覺區分明顯
✓ 避免文化偏見（紅≠壞）
✓ 考慮色盲友好

建議色板（8色）：
#4e79a7 - 藍
#f28e2c - 橙
#e15759 - 紅
#76b7b2 - 青
#59a14f - 綠
#edc949 - 黃
#af7aa1 - 紫
#ff9da7 - 粉
```

### 連續色彩

```
用於表示數值大小

單色漸變：
低 ────────────────────── 高
#dbeafe  #93c5fd  #3b82f6  #1d4ed8

發散色彩（有中點）：
負 ─────────── 0 ─────────── 正
#ef4444  #fca5a5  #fff  #86efac  #22c55e
```

### 色盲友好

```
紅綠色盲最常見（8% 男性）

避免：
✗ 僅用紅/綠區分（如好/壞）
✗ 相似明度的紅綠相鄰

替代方案：
✓ 藍/橙 代替 紅/綠
✓ 搭配圖案或圖示
✓ 使用 ColorBrewer 色板
✓ 測試工具：Coblis, Color Oracle
```

## 品牌色彩選擇

### 步驟

```
1. 了解品牌個性
   ├── 品牌價值觀
   ├── 目標受眾
   └── 競爭對手分析

2. 選擇主色
   ├── 符合產業慣例或差異化
   ├── 考慮心理聯想
   └── 確保可用性（對比度）

3. 建立色板
   ├── 主色變體（50-900）
   ├── 輔助色
   ├── 中性色
   └── 功能色（成功、錯誤等）

4. 測試驗證
   ├── 不同裝置顯示
   ├── 深淺模式
   ├── 無障礙測試
   └── 列印效果
```

### 主色變體生成

```css
:root {
  /* 基於主色生成變體 */
  --primary-50: #eff6ff;   /* 最淺 - 背景 */
  --primary-100: #dbeafe;
  --primary-200: #bfdbfe;
  --primary-300: #93c5fd;
  --primary-400: #60a5fa;
  --primary-500: #3b82f6;  /* 基礎色 */
  --primary-600: #2563eb;  /* 主色 - 按鈕 */
  --primary-700: #1d4ed8;  /* Hover */
  --primary-800: #1e40af;
  --primary-900: #1e3a8a;  /* 最深 - 文字 */
  --primary-950: #172554;
}
```

## CSS 色彩現代語法

### OKLCH 色彩空間

```css
/* OKLCH - 更感知均勻的色彩空間 */
:root {
  /* oklch(lightness chroma hue) */
  --primary: oklch(59% 0.2 255);      /* 藍色 */
  --primary-light: oklch(75% 0.15 255);
  --primary-dark: oklch(45% 0.2 255);
}

/* 優點：調整明度時色相不偏移 */
```

### Color-mix 函數

```css
:root {
  --primary: #2563eb;

  /* 混合產生變體 */
  --primary-light: color-mix(in oklch, var(--primary), white 30%);
  --primary-dark: color-mix(in oklch, var(--primary), black 20%);

  /* 透明度變體 */
  --primary-alpha: color-mix(in srgb, var(--primary), transparent 50%);
}
```

### 相對色彩語法

```css
:root {
  --primary: #2563eb;

  /* 基於主色調整 */
  --primary-lighter: oklch(from var(--primary) calc(l + 0.2) c h);
  --primary-darker: oklch(from var(--primary) calc(l - 0.15) c h);
  --primary-desaturated: oklch(from var(--primary) l calc(c - 0.1) h);
}
```
