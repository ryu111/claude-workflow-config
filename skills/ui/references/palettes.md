# 產品配色方案庫

96+ 按產業分類的配色方案，包含完整的色彩系統。

---

## 使用方式

1. 根據產品類型找到對應分類
2. 選擇適合品牌調性的方案
3. 複製 CSS 變數到專案
4. 根據需要微調

## 深色模式變體

### 通用深色轉換規則

```css
/* 淺色 → 深色轉換 */
:root[data-theme="dark"] {
  /* 背景：反轉 */
  --color-background: #0f172a;
  --color-surface: #1e293b;

  /* 文字：反轉 */
  --color-text: #f1f5f9;
  --color-text-muted: #94a3b8;

  /* 主色：飽和度降低 10%，亮度提高 */
  --color-primary: #60a5fa;  /* 原 #2563eb */

  /* 邊框：變暗 */
  --color-border: #334155;

  /* 功能色：微調 */
  --color-success: #4ade80;
  --color-warning: #fbbf24;
  --color-error: #f87171;
}
```

## 自定義指南

### 從品牌色開始

```css
/* 1. 確定品牌主色 */
--brand-color: #2563eb;

/* 2. 生成色階 */
--color-50: /* 最淺 */;
--color-100: /* ... */;
--color-500: var(--brand-color);
--color-900: /* 最深 */;

/* 3. 映射到語意色 */
--color-primary: var(--color-600);
--color-primary-hover: var(--color-700);
--color-primary-light: var(--color-100);
```

### 對比度檢查

```
WCAG AA 標準：
- 正常文字：4.5:1
- 大文字（18px+）：3:1
- UI 元件：3:1

工具：
- WebAIM Contrast Checker
- Coolors Contrast Checker
```
